LIBRARY MANAGEMENT SYSTEM


The LMS is a web application designed to streamline library operations. Admins can manage books, users, and issue/return records efficiently. Users can browse available books and view their issue status through a simple interface. Developed a web-based Library Management System as a team of two. I was responsible for designing the User Interface (UI) and developing the backend APIs using Node.js and Express.js.

- Designed a responsive and user-friendly frontend interface to manage books, users, and transactions.
- Built RESTful APIs to handle core functionalities like user authentication, book issuing/returning, and data management.
- Integrated the frontend with backend services to enable smooth and dynamic user interactions.
- Ensured modular and scalable code structure for future enhancements.


Language Used : HTML5, CSS3, JS, EJS, BOOTSTRAP5, Node.js, Express.js, MySQL

GitHub link : https://github.com/ketanmohate/Library-Management-System
